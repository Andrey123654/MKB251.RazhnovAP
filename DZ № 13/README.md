# Лабораторная работа: Взаимодействие с API VirusTotal v3

## 📋 Описание проекта
Данный проект представляет собой Python-скрипт для взаимодействия с API VirusTotal v3. Скрипт позволяет выполнять различные операции по анализу файлов, URL, доменов и IP-адресов, а также отслеживать историю активности.

## 🎯 Цель работы
- Научиться взаимодействовать с API средств защиты информации (СЗИ)
- Освоить авторизацию по API, выполнение запросов и обработку ответов в формате JSON

## 🚀 Функциональность

### Основные функции:
| № | Функция | Описание |
|---|---------|----------|
| 1️⃣ | Информация о файле | Получение информации о файле ELF file по SHA-256 хэшу |
| 2️⃣ | Информация о сайте | Проверка сайта rtmis.ru в базе VirusTotal |
| 3️⃣ | Информация о домене | Анализ домена rtmis.ru |
| 4️⃣ | Информация об IP-адресе | Проверка IP-адреса 85.142.23.34 |
| 5️⃣ | Отправить файл на анализ | Загрузка ELF file на VirusTotal |
| 6️⃣ | Отправить URL на анализ | Отправка https://rtmis.ru на анализ |
| 7️⃣ | Комментарии к файлу | Просмотр комментариев сообщества |
| 8️⃣ | Голосование по файлу | Просмотр результатов голосования |
| 9️⃣ | Повторный анализ файла | Запрос повторного анализа |
| 🔟 | Проверить статус анализа | Проверка статуса по ID анализа |
| 1️⃣1️⃣ | История активности | Комплексный просмотр истории из API |
